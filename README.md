# KaryaNexa

**KaryaNexa** is a modern, technology-driven web application designed to evaluate and rank employee performance objectively using the **TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)** method. Built with the **Laravel framework**, KaryaNexa provides a structured, transparent, and scalable solution for organizations seeking data-driven performance appraisal systems.

## üöÄ Overview

Traditional employee performance evaluations often suffer from subjectivity and inconsistency. KaryaNexa addresses this problem by implementing a **multi-criteria decision-making (MCDM)** approach, ensuring that performance assessments are **measurable, fair, and scientifically grounded**.

KaryaNexa is not an API service. It is a **full-featured web application** that includes user interfaces, business logic, and database integration, all developed within the Laravel ecosystem.

## üéØ Key Features

- **Employee Performance Evaluation**  
  Assess employees based on multiple criteria such as discipline, productivity, responsibility, and other customizable metrics.

- **TOPSIS-Based Ranking Engine**  
  Automatically calculates normalized values, weighted matrices, ideal solutions, and final preference scores to produce accurate rankings.

- **Dynamic Criteria & Weight Management**  
  Easily configure evaluation criteria and their respective weights without modifying core logic.

- **Role-Based Access Control**  
  Designed to support roles such as Admin, HR, and Evaluator for secure and structured access.

- **Clear & Explainable Results**  
  Ranking results are transparent and traceable, making them suitable for academic, institutional, and professional use.
